# Metaorder modelling and identification

### University of Cape Town
### Statistics and Data Science Masters thesis
### Ezra Goliath (GLTEZR001)

## Overview



This repository contains the python code used to implement the algorithms that generate the synthetic metaorders and verify the Lillo-Mike-Farmer (LMF) order-splitting theory for Metaorder Modelling and Identification Masters thesis by Ezra Goliath. The repository also contains all the code for data processing, verification of the metaorders and generation of plots and tables. 

# Repository structure:

Data: contains code for processing raw data and outputting .csv files used for the metaorder generation, plots and tables

Plots: contains the code for processing .csv files and outputting plots
